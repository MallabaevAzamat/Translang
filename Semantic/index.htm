Семантика

Сема́нтика (от др.-греч. σημαντικός — обозначающий) — раздел языкознания, изучающий значение единиц языка. (Из Википедии)

Базовой структурой является сборка, она включает в себя набор элементов сборки
на Nemerle описание её ячейки AST выглядит так:
symbol module Assembly
{
 Items: list[AssemblyItem] {get; set;}
}
 abstract symbol module AssembolyItem { }
Следующим элементом является функция - рабочая лошадка программы:
symbol module Function: AssemblyItem
{
 ...
}
